# Blog-Website
Django

## Installation
Python and Django need to be installed <br/>
`pip install django`

## Usage
Go to project folder and run <br/>
`python manage.py runserver` <br/>
Then go to the browser and enter the url [http://127.0.0.1:8000](http://127.0.0.1:8000/).
